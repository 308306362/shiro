Provides the JSecurity JSP Tag Library implementations.

JSecurity JSP Tags can be used to evalute or not evaluate (show or not show) parts of a JSP page based on the current user's authentication status and/or authorization (access control) abilities.

The classes of most interest are:

  • AuthenticatedTag
  • NotAuthenticatedTag
  • HasRoleTag
  • LacksRoleTag
  • HasPermissionTag
  • LacksPermissionTag

The other classes in this package are usually superclasses of those above for consolidating common behavior.
